# Date Filter Usage Guide - Live Route Tracking

## Quick Access Steps

### 1. Navigate to Route Tracking
- Go to **Live Tracking** tab in attendance management
- Click the **purple button**: "View Route Tracking & Playback"
- This opens the full LiveTrackingScreen with three tabs

### 2. Access Historical Routes
- Click the **"Historical Routes"** tab (third tab)
- You'll see the date filter interface

### 3. Use Date Filter
```
Historical Routes - Dec 23, 2024    [📅]
┌─────────────────────────────────────────┐
│ Select Salesman: [All Salesmen ▼]      │
└─────────────────────────────────────────┘
[Load Routes]
```

### 4. Filter Options
- **📅 Calendar Icon**: Click to select different date (past 30 days)
- **Employee Dropdown**: Choose specific salesman or "All Salesmen"
- **Load Routes Button**: Refresh data with current filters

## What You'll See

### Map Markers
- 🏠 **Purple Markers**: Home locations (where work started)
- ▶️ **Green Markers**: Punch-in locations
- ⏹️ **Red Markers**: Punch-out locations
- **Blue Dashed Lines**: Route paths traveled

### Route Summary Cards
- Employee name and working hours
- Start and end times
- Total distance (if available)

## Example Workflow

1. **Select Yesterday**: Click 📅 → Choose yesterday's date
2. **Filter by Employee**: Select "John Doe" from dropdown
3. **View Results**: See John's route with home location marked
4. **Toggle Views**: Use app bar icons to show/hide routes/home locations
